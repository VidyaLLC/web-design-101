Vidya Tutorial: Web Design 101

The relationship between HTML, CSS, and JavaScript

HTML, CSS, and, JavaScript are essential technologies for any website. They all do different things, but they blend together nicely to create a great experience for your users.

HTML

HTML defines the structure of a web page. It has all the "stuff" in it: text, images, links, textboxes, buttons, dropdowns, etc. Just dump it all into the page to get started.

CSS

CSS makes it all pretty. It defines everything about the look of every element: colors, dimensions, spacing, fonts, etc. If you're really clever, you can use CSS to do some basic animations and other effects. This tutorial uses some CSS from the awesome Twitter Bootstrap project.

JavaScript

JavaScript gives your page life. It is a programming language where you can write code, or use other people's code, to make your page do cool things. This tutorial uses the beloved JQuery library to help the button do some work.
